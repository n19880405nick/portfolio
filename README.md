# <幸せを呼ぶ昼寝>

## サイト概要
### サイトテーマ
昼寝をして幸せな気持ちになれたエピソードなどの投稿や検索、
<br>
自分がいつどのくらい昼寝できたのかの確認が出来るコミュニティサイト

### テーマを選んだ理由
自分自身が昼寝をすることで、頭が冴えたり、良いことにつながることが多かったため
<br>
また、昼寝について説明のサイトは多くあっても、コミュニティサイトはあまり見受けられなかったため

### ターゲットユーザ
主に社会人

### 主な利用シーン
夜、その日を振り返る時に
